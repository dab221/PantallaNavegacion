Navegación entre pantallas con Jetpack Compose

Descripción

Este proyecto es una aplicación de práctica en Kotlin utilizando Jetpack Compose y Navigation Compose. Su objetivo es aprender los conceptos fundamentales de la navegación
entre múltiples pantallas en Android moderno.

La palicación cuenta con tres pantallas:

-Inicio
-Cliente
-Perfil

A través de botones, el usuario puede navegar entre ellas utilizando un NavController, mientras que el NavHost administra qué pantalla se muestra en cada
momento.

Conceptos practicados

-Navigation Compose
-NavController
-NavHost
-composable
-Rutas (Routes)
-Navegación entre múltiples pantallas
-popBackStack() para regresar a la pantalla anterior
-Organización del proyecto en archivos y pantallas independientes

Tecnologias utilizadas

-Kotlin
-Jetpack Compose
-Navigation Compose

Objetivo

Comprender cómo implementar la navegación en aplicaciones Android modernas utilizando una única Activity u múltiples pantallas (Composable),
siguiento las buenas prácticas recomendadas por Google.
